Reflective Site Report

Building this portfolio has been an eye-opening journey into the world of web development. When I first started, I thought creating a website required complex tools, but I quickly realized how much power lies in pure HTML5 and CSS3. The learning process was steady; I began by understanding the importance of semantic tags like <header> and <nav>, which not only help with organization but also improve accessibility and SEO.

The coding process involved a lot of trial and error. One of my biggest challenges was getting the CSS Grid to work correctly on the projects page. Initially, the images were overlapping, but after learning about grid-template-columns: repeat(auto-fit, ...), I was able to make the layout fluid and responsive. Debugging the navigation bar also took some time, especially ensuring it looked good on mobile devices using Media Queries.

For the UI design, I chose a slate and blue color palette to maintain a professional yet modern feel. I opted for a sans-serif font stack to ensure readability across different screens. My main influences were minimalist design portfolios like those found on Behance and Awwwards, where "less is more."

Looking back, my progress has been significant. I went from barely knowing how to link a CSS file to creating a multi-page, responsive site. The success of seeing a form work and a grid snap into place on a phone screen was very rewarding. In the future, I hope to add even more complex layouts and perhaps dive into CSS animations.
